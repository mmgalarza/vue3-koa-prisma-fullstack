# Stack Tecnológico: Terminal de Ventas Kiosko

## Frontend Core
- **Framework:** React 19 (o última versión estable)
- **Lenguaje:** TypeScript (Strict Mode)
- **Herramienta de Construcción:** Vite

## UI & Estilos
- **Librería de Componentes:** Material UI (MUI) - Versión gratuita
- **Iconos:** @mui/icons-material

## Gestión de Estado y Datos
- **Estado Global:** React Context API (Nativo)
- **Comunicación API:** Fetch API Moderno
- **Validación de Esquemas:** Zod (para asegurar contratos de API)

## Backend (Existente)
- **Runtime:** Node.js
- **Framework:** Koa.js
