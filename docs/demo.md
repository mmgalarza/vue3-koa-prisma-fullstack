# Descripción corta (CV / LinkedIn)

## Versión 1 (1–2 líneas)
**Sistema de gestión full‑stack** con SPA en Vue 3 y API en Koa + Prisma. Incluye autenticación JWT, CRUD de productos/clientes/operaciones, documentación con Swagger y base de datos PostgreSQL.

## Versión 2 (3–4 líneas)
Proyecto full‑stack que integra un frontend SPA en Vue 3 (Vite, Vuetify, Pinia) y un backend en Koa + TypeScript con Prisma y PostgreSQL. Implementa autenticación JWT, activación por email, recuperación de contraseña y un set completo de CRUDs (usuarios, clientes, categorías, productos y operaciones). Incluye validaciones con Zod y documentación OpenAPI/Swagger.

## Bullet points (para CV)
- Desarrollo de SPA con Vue 3, Vuetify, Pinia y Vue Router.
- API REST con Koa + TypeScript, Prisma ORM y PostgreSQL.
- Autenticación JWT, activación de cuenta y reset de contraseña por email.
- Documentación de endpoints con Swagger/OpenAPI.
- Arquitectura modular por dominios y contratos compartidos.

## Tecnologías
Vue 3, Vite, Vuetify, Tailwind, Pinia, Vue Router, Koa, TypeScript, Prisma, PostgreSQL, Zod, JWT, Swagger

## Palabras clave (ATS)
Full‑stack, SPA, REST API, TypeScript, Prisma, PostgreSQL, JWT, OpenAPI, Swagger, Vue
